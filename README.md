# Trading Behavior Analysis: Market Sentiment Impact

## Overview
Analysis of how trader behavior correlates with Bitcoin market sentiment using Fear & Greed Index data.

## Project Structure
ds_Arush_U_Rai/
├── notebook_1.ipynb # Google Colab analysis notebook
├── csv_files/ # Data files
│ ├── Historical Data.csv # Raw trader data
│ └── Fear Greed Index.csv # Sentiment data
├── outputs/ # Visualizations
│ └── Comprehensive Sentiment Analysis from Colab.png
├── ds_report.pdf # Final analysis report
└── README.md # This file

## Key Findings
- Fear periods drive 5x higher trading volume but 40% lower win rates
- Extreme Greed periods show highest profitability (46.7% win rate)
- Counter-sentiment strategies demonstrate superior performance
